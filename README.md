# AI CV Reviewer

An AI-powered CV reviewer that analyzes your CV text, gives a score out of 100, and provides detailed feedback on strengths, weaknesses, and suggestions for improvement. Optionally specify a target position for role-specific feedback.

## Live Demo

[Try it here](https://bunyaminkeremon.github.io/ai-cv-reviewer/)

## What It Does

- Paste CV text and get an AI-generated score (0-100)
- Detailed feedback on strengths and weaknesses
- Specific improvement suggestions
- Missing information detection
- Optional target position for role-fit analysis
- Color-coded score indicator

## Tech Stack

- HTML5
- CSS3
- JavaScript (async/await, Fetch API)
- Google Gemini API for AI analysis

## Run Locally
```bash
git clone https://github.com/bunyaminkeremon/ai-cv-reviewer.git
```

Open `index.html` in your browser. You'll need your own API key from aistudio.google.com — replace the key in `script.js`.
